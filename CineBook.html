About CineBook

A seat-locking system for modern premium cinemas.

🏛️

5 Premium Halls

Halls A–E each seat 80+ guests across 5 distinct pricing zones — from near-screen economy to Royal Suite recliners at the back.

🔒

Seat Locking

Once booked, seats are immediately locked and persist across sessions via browser storage — just like a real booking system.

🎞️

Multi-Format

5 films, multiple languages, 2D or 3D — every combination bookable with transparent pricing before you pay.

⚙️

System Functions

  • Show() — Initialise 80-seat hall
  • load() / save() — Persist seat state
  • display() — Render visual layout
  • isAvailable() — Check seat status
  • book() — Lock selected seats
  • available() / booked() — Live counts
  • bookSeats() — Full booking flow
  • search() — Retrieve by ID
  • report() — Occupancy stats
  • menu() — Navigation system
